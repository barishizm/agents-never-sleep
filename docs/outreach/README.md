# Outreach — proposals only

Everything in this directory is a **proposal for the maintainer**. Nothing here is published,
applied, or sent anywhere automatically — not by an unattended run, not by CI. Publication and
repo-settings changes are explicit maintainer actions.

| file | what it proposes | who acts |
|---|---|---|
| [github-metadata.md](github-metadata.md) | repo description, topics, social preview, release notes (+ the exact `gh` commands) | maintainer applies |
| [proposals.md](proposals.md) | technical article proposals (deep dives, never ads) | maintainer selects & gates |
| [terminology-memo.md](terminology-memo.md) | category-name research: "Execution Governance" vs "Operational Governance" | maintainer decides |
